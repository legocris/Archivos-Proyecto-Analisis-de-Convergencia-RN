# Archivos-Proyecto-Analisis-de-Convergencia-RN
Archivos usados para hacer el Proyecto Integrador de Analisis de Convergencia de Redes Neuronales Angostas contra Profundas usando el perceptrón para problemas de clasificación. El proyecto fue entregado como Proyecto Integrador de Modelación Matemática en la Licenciatura en Matemáticas. CUCEI. UDG.
